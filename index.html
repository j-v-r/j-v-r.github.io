Metro Info App

This project is a work in progress. The project demonstrates my ability to use Ajax technology with thrid party APIs to develop web applications. In this project users can browse information from 5 different US cities. When a city button is clicked three widgets will display current weather, traffic conditions, and Flickr photos. This project uses the jQuery and Leaflet.js JavaScript libraries. The 3rd party APIs used are Weather Underground, MapBox, and Flickr.
